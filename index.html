Projects

Tech Art

Batch Animations UI Tool (Python | MEL | PySide2)

This script provides the user with a User Interface tool that allows them to batch a series of animation files onto a character rig and save the connected animations to a specified directory

Export Camera as FBX UI (Python | PySide2)


This script provides a UI that allows a user to export a camera as a fbx file in the desired path. The user setup adds a dropdown menu to the Maya menu bar. Clicking on the menu calls the fbx_export_camera file and instantiates a UI

Asset Importer with Material Application (Python | UE4 Blueprints)

This tool provides a UI that allows the user to import a folder of fbx assets into the project. The imported assets will automatically have materials applied to them if their material slot names match with materials already existing in the project. Additionally, the user has a choice of having the tool potentially catch small misspellings by applying the closest-spelled material to the asset if the material slot doesn't exactly match any existing materials

Animation Importer (Python | UE4 Blueprints)

This tool provides a UI that allows the user to input a shot folder of FBX character animations, a skeletal mesh, and the destination where the applied animations will live. It then calls a python script that applies the animations to the mesh and individually imports each applied animation into the working project
